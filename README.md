# Single Sign On

Use for Login in Internet UGM

## Installation

1. Install Python3
2. pip install -r requirements.txt

## How To Use

### Login with input

1. python3 login.py
2. enter your UGM ID (without @\*.ugm.ac.id)
3. enter your Password

### Autologin

1. edit config.json
2. enter your UGM ID (without @\*.ugm.ac.id)
3. enter your Password
4. make a service to run autologin.py when startup
